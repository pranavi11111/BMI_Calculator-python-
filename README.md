# BMI Calculator (Python)

A simple Python program to calculate Body Mass Index (BMI) using user input for height and weight.

---

##  Features
- Takes user input (height & weight)
- Calculates BMI instantly
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

---

## Tech Stack
- Python 3

---

## BMI Formula

BMI = weight (kg) / (height (m))²
